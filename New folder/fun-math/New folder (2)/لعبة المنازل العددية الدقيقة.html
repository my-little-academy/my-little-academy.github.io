لعبة المنازل العددية الدقيقة

تعلم المنازل العددية باستخدام المربعات والقضبان والصفائح

الوحدات
كل مربع = 1
العدد: 0
العشرات
كل قضيب = 10 مربعات
العدد: 0
المئات
كل صفيحة = 100 مربع
العدد: 0
الآلاف
كل مكعب كبير = 1000 مربع
العدد: 0

كيفية اللعب:

1. أدخل رقماً بين 0 و 9999 ثم اضغط "اظهر العدد"

2. شاهد تمثيل العدد بالمربعات والقضبان والصفائح

3. لاحظ العلاقة بين المنازل:

- 10 مربعات = 1 قضيب

- 10 قضبان = 1 صفيحة (100 مربع)

- 10 صفائح = 1 مكعب كبير (1000 مربع)

4. جرب أرقاماً مختلفة لفهم الفكرة بشكل أفضل
