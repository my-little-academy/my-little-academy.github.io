نماذج تمارين المقاطع الصوتية

1. لعبة سحب الحروف لتكوين مقطع

سَ ا بَ و رُ ي
اسحب الحروف هنا لتكوين مقطع

2. توصيلة المقاطع

صل المقطع بالكلمة المناسبة:

  • مقطع: با ⟶ ( ) باب / كتاب / بطة
  • مقطع: كي ⟶ ( ) كِتاب / كيس / كيكة

3. اختيار عدد المقاطع

كم عدد المقاطع في كلمة "مُعَلِّمَةٌ"؟

4. اسحب الكلمات إلى جدول المقاطع

اسحب كل كلمة إلى الخانة المناسبة حسب عدد المقاطع:

شَجَرَة قَمَر الْأَرْضُ
كلمة بها 2 مقطع كلمة بها 3 مقاطع

5. مشهد الشمس والقمر

في قصر الشمس، تُخفى اللام خلف الحرف المشدد. أما في قصر القمر، تظهر اللام مع همزة الوصل.

حدد نوع "ال" في الكلمات التالية:

  • الشَّمْس: [ ] شمسية [ ] قمرية
  • القَمَر: [ ] شمسية [ ] قمرية
